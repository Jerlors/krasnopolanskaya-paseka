# Краснополянская пасека | Горный мёд

Интернет-магазин натурального горного мёда с альпийских лугов.

## 📁 Структура проекта

```
├── index.html          # Главная страница
├── catalog.html        # Каталог продукции
├── product.html        # Страница товара
├── cart.html           # Корзина
├── about.html          # О пасеке
├── delivery.html       # Доставка
├── contacts.html       # Контакты
├── css/                # Стили
│   ├── variables.css   # CSS-переменные
│   ├── reset.css       # Сброс стилей
│   ├── layout.css      # Макет
│   ├── components.css  # Компоненты
│   └── pages.css       # Стили страниц
├── js/                 # JavaScript
├── img/                # Изображения
├── data/               # Данные
└── README.md           # Этот файл
```

## 🚀 Быстрый старт

1. Откройте `index.html` в любом современном браузере
2. Или используйте локальный сервер:
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js (если установлен http-server)
   npx http-server
   ```

## 🛠 Технологии

- HTML5
- CSS3 (с переменными)
- Ванильный JavaScript
- Google Fonts (Comfortaa, Inter)

## 📦 Функционал

- Просмотр каталога мёда
- Детальная страница товара
- Корзина покупок
- Адаптивный дизайн (мобильная и десктопная версии)
- Навигация по разделам сайта
